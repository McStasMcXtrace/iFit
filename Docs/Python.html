This feature is experimental.

Requirements: Python/NumPy

Install Python(x,y) for Windows, Anaconda or Entought Canopy for Windows and MacOSX. Linux flavors provide numpy as e.g. RPM and Debian packages.The IPython package is also recommended.

The required packages are:
  • python
  • numpy
  • scipy
  • hdf5storage
which can all be installed on Ubuntu/Debian systems using our http://packages.mccode.org repository with:
  • sudo apt-get install python-hdf5storage

Usage

Navigate to the iFit/Applications/Python directory and launch IPython or Python. You may even use IPython notebook.
Then issue the command:

>>> from matlab import Matlab
>>> mlab = Matlab()
>>> mlab.open()
or, when using the standalone version:
>>> mlab.open('ifit')

From there on, any iFit and Matlab expression can be executed from Python.

>>> a = mlab.eval("a=iData('filename')")
>>> mlab.eval("plot(a)")
>>> mlab.eval("b = a/2")
>>> b=mlab.get('b')
The main idea is that any call to iFit/Matlab should be prepended by 'mlab.' and use the set/get/eval methods.

iFit objects are stored as Python NumPy arrays. Refer to the Methods page to learn about all methods that can be called for iData and iFunc objects.

In addition, there are specialized methods to communicate with iFit/Matlab:
mlab.set('name', value)
assigns name=value in the iFit/Matlab workspace.
mlab.get('name')
retrieves the named variable from the iFit/Matlab workspace.
mlab.eval("expression")
evaluates the expression/code in the iFit/Matlab workspace.
