iFit: Changes


  • 1.0: fixed minor error in looktxt and compatibility issues with Windows. First official release.
  • 1.0-RC11: pre-release. Added FFT, convolution, Laplacian, gradient and Jacobian in Math.
  • 1.0-RC10: added uncertainty estimate on fit parameters for iOptim methods and iData/fits.
  • 1.0-RC9: fixed value for Monitor in binary/combine. Clarify handling of axis 0 as Signal/Monitor.
  • 1.0-RC8: provide MeX pre-compiled files; fixed a few bugs in looktxt/texmex/iLoad. ifitmakefunc tool to create fit functions (June 2011)
  • 1.0-RC7: fixed iFit/mcstas (Windows support), added more optimizers in iOptim (May 2011)
  • 1.0-RC6: fixed import of multicolumn text files
  • 1.0-RC5b: fixed typo in looktxt.
  • 1.0-RC5: added self test procedure 'ifittest' and fixed a set of bugs
  • 1.0-RC4: added McStas optimization wrapper
  • 1.0-RC3: fixed multiple files import in iLoad and DAT export in class2str (March 2011)
  • 1.0-RC2: fixed iLoad error for e.g. ILL TAS import (Feb 2011)
  • 1.0-RC1: initial release (Feb 2011)
