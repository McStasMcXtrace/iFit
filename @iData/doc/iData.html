Usage

The iData Matlab class (for Matlab version >= 6) is a multipurpose object which is designed to import any data file/variable, to process it using mathematical operations, to fit it to models, to view it, and to save it into various standard formats. The notions of iData aliases and axes are of major importance to master the iData operations.

Object capabilities include:
  • loading of data files/variables (any text files or specific load routines), maintaining the data structure and labeling
  • easy access to any part of the data file, define aliases (aka links/shortcuts)
  • easy axes definition to use when applying mathematical operations
  • all standard mathematical operations
  • plotting methods
  • fitting/optimization methods
  • saving results
