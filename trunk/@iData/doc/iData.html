iData Matlab object class

The iData Matlab class (for Matlab version >= 6) is a multipurpose object which capabilities include:
  • loading of data files (any text files or specific load routines), maintaining the data structure and labeling
  • easy access to any part of the data file, define aliases (aka links/shortcuts)
  • easy axes definition to use when applying mathematical operations
  • all standard mathematical operations
  • plotting methods
  • fitting/optimization methods
  • saving results
