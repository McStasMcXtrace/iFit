Requirements

The only requirement is to have Matlab installed. Any Matlab version from 6.5 (2005) will do.
This is a commercial product. Student versions can be bought for lower prices, as well as Academic licenses (e.g. for Universities). A C compiler is often shipped with Matlab (and installed automatically). The iFit library may use it sporadically.

Obtaining the package

The package is currently available from the mccode.org web site
  • source package for all platforms, requires a Matlab license to run [ zip, 3.1 Mb ].
  • standalone packages for Linux, Windows and Mac OS X will be available in future versions.

Installation

You just need to copy the iFit directory somewhere, say in /path/to.
  1. Extract first the iFit archive if needed.
  2. Just drag-drop the iFit directory in the target location that suits you.
For our users who love challenges and command lines, this may be achieved e.g. with
% unzip iFit.zip
% cp -r iFit /path/to
Start Matlab by clicking on its icon or typing the matlab command in a terminal.
% matlab
In order to use iFit, you need to import all its methods into Matlab each time you start it (this may be automated from the startup.m file)
>> addpath(genpath('/path/to/iFit'))
That's all !
