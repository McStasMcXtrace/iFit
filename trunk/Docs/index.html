iFit:



These objects have been defined in order to perform the following operations seamlessly:
  • Load any text formated file, and a large set of binary format files
  • Display object information, including metadata from the original file (see below)
  • Plot the object Signal in a simple view for 1D, 2D and 3D objects
  • Perform mathematical operations using objects (=-/* trigonometric projections integration re-binning...
  • Fit a parametrized model to the object Signal
  • Save the object into a set of output formats
